# Contributing Guidelines

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make meaningful commits
4. Submit a pull request

## Commit Message Format

- `feat:` New features
- `fix:` Bug fixes  
- `docs:` Documentation updates
- `refactor:` Code refactoring
- `test:` Adding tests
